# Campeonato Baby Fútbol 2026

Sitio web dinámico para el seguimiento de la Liga de Baby Fútbol 2026.

## Características
- **Sincronización en tiempo real**: Los datos se cargan directamente desde un Google Sheets.
- **Tablas Automáticas**: Cálculo de posiciones para Apertura, Clausura y Tabla Anual.
- **Estadísticas Avanzadas**: Goleadores, vallas menos vencidas, balance de goles y efectividad.
- **Diseño Cyber/Neon**: Interfaz moderna con vidrio (glassmorphism) y efectos de luz.
- **Gestión de Clubes**: Integración de escudos de los 9 equipos participantes.

## Tecnologías
- HTML5 / CSS3 (Vanilla)
- JavaScript (ES6+)
- Google Sheets API (CSV)

## Instalación Local
Simplemente descarga el repositorio y abre `index.html` en tu navegador. 

*Nota: Requiere conexión a internet para cargar los datos del torneo.*
